## 版本详细说明

### v1.0.4 - 实时速度监控 ✅

**核心功能**:
- 种子列表实时速度显示（10秒刷新）
- 下载器仪表盘实时速度
- 速度历史图表（最近1小时）
- 动静数据分离，减少API调用

### v1.0.9 - 一键部署 ✅

**核心功能**:
- PyInstaller 打包为单可执行文件（前后端合一）
- Inno Setup Windows 安装包（.exe 安装向导）
- fpm Linux 安装包（.deb/.rpm）
- Docker Compose 全栈部署
- systemd 服务配置
- GitHub Release 自动发布

**部署方式**:
- Docker: `docker compose up -d`
- Windows: 下载安装包运行
- Linux: `dpkg -i btdeck.deb` 或 `rpm -i btdeck.rpm`

## 技术栈

### 后端
- **Python**: 3.11+
- **FastAPI**: 0.115.0
- **SQLAlchemy**: 2.0.15
- **数据库**: SQLite / PostgreSQL

### 前端
- **Vue**: 2.6.12
- **TypeScript**: 4.x
- **Element UI**: 2.15.13

### 部署
- **Docker**: docker-compose 全栈部署
- **打包**: PyInstaller (可执行文件) + Inno Setup (Windows) + fpm (Linux)
- **服务管理**: systemd (Linux) / Windows Service
